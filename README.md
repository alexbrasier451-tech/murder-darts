# Darts Night Scorer

A mobile-first installable PWA for darts scoring.

## Games

- Murder
- X01, including 301/501/custom starts, stats-only inferred double-in tracking, best-of legs, race-to legs, and race-to sets

## Features

- Main menu with resumable games
- Stored player names with add/edit/delete
- Offline-capable PWA install
- Mobile-first scoring controls
- Undo for live games
